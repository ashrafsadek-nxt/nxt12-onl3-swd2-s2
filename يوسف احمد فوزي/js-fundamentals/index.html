Welcome to JavaScript

What is JavaScript?

JavaScript is a versatile programming language that powers the modern web. It can:

  • Make websites interactive
  • Handle user events (clicks, form submissions, etc.)
  • Manipulate HTML and CSS dynamically
  • Communicate with servers
  • Build complete web applications

Variables - Storing Data

What are Variables?

Variables are containers that store data values. Think of them as labeled boxes where you can put different types of information.

Variable Declaration Methods:

// Modern way (recommended)
let userName = "Youseif";
const PI = 3.14159;

// Older way (avoid in modern code)
var oldVariable = "legacy";

Conditions - Making Decisions

Conditional Statements

Conditions allow your code to make decisions based on different scenarios. 

if (age >= 18) {
    console.log("You can vote!");
} else if (age >= 16) {
    console.log("You can drive!");
} else {
    console.log("You're still young!");
}

Loops - Repeating Actions

Types of Loops

For Loop

for (let i = 0; i < 5; i++) {
    console.log(i);
}

While Loop

let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

Functions - Reusable Code Blocks

What are Functions?

Functions are reusable blocks of code that perform specific tasks.

Function Declaration:

function greetUser(name) {
    return "Hello, " + name + "!";
}

// Arrow function (modern syntax)
const greetUser = (name) => {
    return `Hello, ${name}!`;
};

Arrays - Lists of Data

Working with Arrays

Arrays store multiple values in a single variable.

let fruits = ["apple", "banana", "orange"];
fruits.push("grape");        // Add to end
fruits.pop();               // Remove from end
fruits[0];                  // Access first item
fruits.length;              // Get array size

Objects - Complex Data Structures

Understanding Objects

Objects store data in key-value pairs, representing real-world entities. 

let person = {
    name: "John",
    age: 30,
    city: "New York",
    greet: function() {
        return "Hello, I'm " + this.name;
    }
};

console.log(person.name);     // Access property
person.age = 31;              // Update property
